# discord-booru-bot

Discord bot to get images from various image boorus. Will try to automatically determine which API is used.

## Usage

Run `python/booru_bot.py <DISCORDTOKEN>` from command line.

## Supported APIs:

* Gelbooru
* Danbooru
* Shimmie2
* e621.net
* Derpibooru
